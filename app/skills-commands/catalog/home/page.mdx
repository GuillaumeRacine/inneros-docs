# Home & Suppliers

Commands for house management, maintenance, and supplier sourcing.

**8 commands** in this category.

| Command | Description |
|---------|-------------|
| `/house-maintenance` | --- |
| `/source-supplier` | Start or resume an autonomous supplier sourcing workflow. |
| `/supplier-add-candidate` | Add a new supplier candidate to the vetting inbox. |
| `/supplier-create` | Create a full supplier entry in the rolodex (typically after selecting a candidate). |
| `/supplier-list-candidates` | List all active supplier candidates by status. |
| `/supplier-post-template` | Generate a posting template for finding suppliers online. |
| `/supplier-search` | Search existing suppliers in the rolodex by trade or name. |
| `/travel-plan` | Start or continue travel research using the travel-researcher agent. |

## Command Details

### /house-maintenance

---

**Usage:** `/house-maintenance [--overdue] [--season]`

---

### /source-supplier

Start or resume an autonomous supplier sourcing workflow.

**Usage:** `/source-supplier <trade> [description]    # Start new sourcing`

---

### /supplier-add-candidate

Add a new supplier candidate to the vetting inbox.

**Usage:** `/supplier-add-candidate <name> <trade> <source>`

---

### /supplier-create

Create a full supplier entry in the rolodex (typically after selecting a candidate).

**Usage:** `/supplier-create <name> <trade>`

---

### /supplier-list-candidates

List all active supplier candidates by status.

**Usage:** `/supplier-list-candidates [--status <status>] [--trade <trade>]`

---

### /supplier-post-template

Generate a posting template for finding suppliers online.

**Usage:** `/supplier-post-template <trade> <brief-description>`

---

### /supplier-search

Search existing suppliers in the rolodex by trade or name.

**Usage:** `/supplier-search [--trade <trade>] [--name <name>] [--rehire <yes|no|any>]`

---

### /travel-plan

Start or continue travel research using the travel-researcher agent.

**Usage:** `/travel-plan [destination or trip name]`

# Communication

Commands for email management, messaging, and notifications.

**6 commands** in this category.

| Command | Description |
|---------|-------------|
| `/gmail-check-respond` | Check recent emails for anything that might need a response but hasn't been labeled. |
| `/gmail-learn` | Run a Gmail learning cycle - analyze recent labeling activity and detect new patterns. |
| `/gmail-optimize` | Review and optimize email management - pattern learning, importance scoring, unsubscribe. |
| `/gmail-suggest-filters` | View pending filter suggestions and apply approved ones. |
| `/media-digest` | Synthesize any content into audio + text: YouTube, articles, podcasts, PDFs, books, Substack, Medium, or local files. |
| `/yasser-report` | Generate a comprehensive weekly report on [USER]'s work by pulling data from all sources. |

## Command Details

### /gmail-check-respond

Check recent emails for anything that might need a response but hasn't been labeled.

**Usage:** `/gmail-check-respond [--days N] [--label ID]`

---

### /gmail-learn

Run a Gmail learning cycle - analyze recent labeling activity and detect new patterns.

**Usage:** `/gmail-learn [--full]`

---

### /gmail-optimize

Review and optimize email management - pattern learning, importance scoring, unsubscribe.

**Usage:** `/gmail-optimize [learn | score | unsubscribe | audit | status]`

---

### /gmail-suggest-filters

View pending filter suggestions and apply approved ones.

**Usage:** `/gmail-suggest-filters [--apply &lt;id>] [--resolve &lt;domain>]`

---

### /media-digest

Synthesize any content into audio + text: YouTube, articles, podcasts, PDFs, books, Substack, Medium, or local files.

**Usage:** `/media-digest [status|process|url &lt;url>|file &lt;path>|book &lt;title>]`

---

### /yasser-report

Generate a comprehensive weekly report on [USER]'s work by pulling data from all sources.

**Usage:** `/yasser-report`

---

# NightCrew Pipeline

NightCrew is a 21-stage nightly batch pipeline that processes the day's accumulated data while the user sleeps. It runs at 2:00 AM via a macOS LaunchAgent and typically completes in 40-60 minutes.

## Stage Details

### Stage 1: File Organization

Scans inbox folders and routes files to their correct PARA locations.

```python
ROUTING_RULES = {
    "*.pdf": {
        "contains_invoice": "2_Areas/Finance/Invoices/",
        "contains_receipt": "2_Areas/Finance/Receipts/",
        "default": "3_Resources/Documents/"
    },
    "*.md": {
        "starts_with_draft": "1_Projects/Writing/Drafts/",
        "default": "00_Inbox/"
    },
    "*.jpg|*.png|*.heic": {
        "screenshot": "3_Resources/Screenshots/",
        "default": "3_Resources/Images/"
    }
}

def route_file(filepath):
    for pattern, rules in ROUTING_RULES.items():
        if matches(filepath, pattern):
            category = classify(filepath, rules)
            move(filepath, rules[category])
            log(f"Routed {filepath} -> {rules[category]}")
```

### Stage 12: Photo Triage

The longest-running stage. Processes new photos from the Photos library:

```python
def triage_photos(since_last_run):
    new_photos = query_photos_db(since=since_last_run)

    for photo in new_photos:
        score = {
            "aesthetic": photo.overall_aesthetic_score,
            "has_faces": photo.face_area_points > 0,
            "is_screenshot": photo.is_detected_screenshot,
            "is_favorite": photo.is_favorite,
        }

        action = classify_photo(score)
        # Actions: keep, review, archive, flag_for_deletion
        log_photo_decision(photo.id, action, score)

    return {
        "processed": len(new_photos),
        "favorites": count(action == "keep"),
        "screenshots": count(action == "archive"),
        "review_needed": count(action == "review"),
    }
```

### Stage 13: Commitment Reconciliation

Cross-references commitments made (from messages, calendar, reviews) against completions:

```python
def reconcile_commitments():
```

## Error Handling

```python
class NightCrewPipeline:
    def run(self):
        results = {}
        for stage in self.stages:
            try:
                result = stage.execute(timeout=stage.timeout)
                results[stage.name] = {"status": "success", "data": result}
            except TimeoutError:
                results[stage.name] = {"status": "timeout"}
                log.warning(f"{stage.name} timed out after {stage.timeout}s")
            except Exception as e:
                results[stage.name] = {"status": "error", "error": str(e)}
                log.error(f"{stage.name} failed: {e}")

                if stage.required:
                    raise PipelineHaltError(stage.name, e)
                # Non-required stages: log and continue

        self.write_run_log(results)
        return results
```

## Adding a New Stage

```python
# 1. Create the stage script
# stages/my_new_stage.py

def execute(config, context):
    """
    Args:
        config: Stage-specific configuration
        context: Results from previous stages
    Returns:
        dict with stage results
    """
    data = fetch_my_data()
    processed = process_data(data)
    return {"items_processed": len(processed), "summary": summarize(processed)}

# 2. Register in nightcrew_config.yaml
# 3. Test with: python nightcrew.py --stage my_new_stage --dry-run
```

# n8n Specialist

Workflow automation expert for n8n CLI, configuration, and integration with InnerOS.

## When to Use

When you need help with n8n workflow automation:

```bash
n8n update:workflow --id=<workflow-id> --active=true
n8n update:workflow --id=<workflow-id> --active=false
n8n update:workflow --all --active=false
```
